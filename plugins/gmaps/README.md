## Manual installation instructions

Requirements:

- Albert - [Installation instructions](https://albertlauncher.github.io/docs/installing/)
    - Albert Python Interface: ``v0.2``

- Python version >= 3.5

Download and run the ``install-plugin.sh`` script or run the following to do
that automatically:

```sh
curl https://github.com/bergercookie/awesome-albert-plugins/blob/master/plugins//gmaps/install-plugin.sh | bash
```
